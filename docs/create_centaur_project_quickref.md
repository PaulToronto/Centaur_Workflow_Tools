## Purpose
Initialize a **Knowledge-First** project structure.

## Usage
```bash
./scripts/create_centaur_project.sh <ProjectName>
```

## V2 Structure
- **`knowledge_base/`**: For learned concepts, notes, and AI summaries.
- **`notebooks/`**: For Jupyter notebooks.
- **`generated_assets/`**: For raw AI exports or temp files.
- **`inputs/` & `outputs/`**: Data flow.

## Drive Sync
Automatically maps to `~/Google Drive/My Drive/Projects/<ProjectName>`.
